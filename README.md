# 🎯 ChurnIQ - Customer Churn Prediction Platform

## 📊 Overview
ChurnIQ is an enterprise-grade customer churn prediction platform built with Streamlit and Python. It uses machine learning to predict customer churn and provides actionable retention recommendations.

## 🚀 Features

### 1. **Upload Dataset**
- Supports CSV and Excel files
- Automatic data validation
- Quick statistics preview

### 2. **Data Exploration**
- Interactive visualizations
- Correlation matrix
- Dataset summary

### 3. **Model Studio**
- Multiple ML/DL algorithms
- Automated model training
- Performance comparison
- Feature importance analysis

### 4. **Real-time Prediction**
- Customer search by ID
- Editable service fields
- Churn risk score gauge
- Retention recommendations
- Personalized risk factors

### 5. **Business Insights**
- Churn rate analysis
- Financial impact assessment
- ROI calculator
- Strategic recommendations

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| Python 3.10+ | Core language |
| Streamlit | Web framework |
| Scikit-learn | Machine learning |
| Pandas | Data manipulation |
| Plotly | Interactive visualizations |
| Matplotlib/Seaborn | Static visualizations |
